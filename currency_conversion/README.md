## Installation
Requirements

 - Windows
 - Python
 - pandas
 -  beautifulsoup4
 - requests
 -  typing
 - lxml

```bash
  $ pip install -r requirements.txt
```
## Run Locally

Go to the project directory

```bash
  cd MyCaseStudy
```

Install dependencies

```bash
  pip install -r requirements.txt
```

To run file, run the following command

```bash
  python Exercise_1_Currency_conversion.py
```

If the Warning.MESSAGE, XMLParsedAsHTMLWarning,
